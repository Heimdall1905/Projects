# BioRag

RAG-пайплайн:
1) Были спарсены научные статьи на тему болезни Альцгеймера(PubMed) 
2) Векторизация фрагментов chroma(Hugging Face embedings)
3) Поиск релеватных документов
4) Вставка в окно контекста модели(Trinity Large Preview (free))
5) Реализации streamlit приложения для удобного пользования

---

Для запуска также требуется создать файл api с ключом(спасибо openrouter.ai)
```bash
streamlit run main.py
```
