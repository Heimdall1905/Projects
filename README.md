# Data Science Portfolio Projects

Репозиторий содержит два проекта в области Data Science и Machine Learning.

## Проекты

### 1. Kaggle: Предсказание цен на дома

**Задача:** Регрессия для предсказания конечной цены домов на основе их характеристик

**Ключевые достижения:**
- Достигнута метрика **RMSE = 0.12** на тестовой выборке
- Реализована комплексная обработка признаков, включая:
  - Обработка пропущенных значений
  - Feature engineering: создание новых признаков на основе существующих
  - Кодирование категориальных переменных
- Построение и сравнение нескольких моделей:
  - Gradient Boosting, CatBoost
  - Random Forest
    
**Технологии:** Python, Pandas, Scikit-learn, CatBoost, matplotlib, seaborn

### 2. Классификация тональности отзывов на фильмы

**Задача:** Многоклассовая классификация тональности текстовых отзывов (3 класса: положительный, нейтральный, отрицательный)

**Ключевые достижения:**
- Достигнута метрика **F1-score (weighted) = 0.85** на тестовой выборке
- Реализована полная NLP- pipeline:
  - Предобработка текста: токенизация, лемматизация, очистка
  - Векторизация текста с использованием BERT-эмбеддингов
  - Работа с дисбалансом классов
- Обучение и тонкая настройка трансформерной модели:
  - Fine-tuning предобученного RuBERT для задачи классификации
  - Оптимизация гиперпараметров модели

**Технологии:** Python, Transformers, PyTorch, Huggingface, NLP, Scikit-learn
